# 字段校验补充说明

上表只描述字段的展示范围，字段的输入校验规则以本节和患者端 PRD 为准。

## 手机号（patient.mobile）

手机号补充支持微信授权和手动输入两种来源，两者的验证要求不同：

- 微信授权带回的手机号：视为已由微信平台完成实名与所有权验证，患者端不重复校验，可直接使用。
- 手动输入的手机号：必须完成两层验证后才允许提交与写入档案。第一层为格式校验，需符合中国大陆手机号格式（11 位、以 1 开头的合法号段），不通过不允许提交；第二层为号码所有权校验，患者需接收短信验证码并回填，验证码校验通过才视为有效。手动输入手机号在未通过上述验证前，不得写入 `patient.mobile`，也不得放行进入后续快速建档、资料补充或接诊关联流程。
