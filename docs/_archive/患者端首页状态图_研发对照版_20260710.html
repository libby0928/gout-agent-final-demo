状态切换规则

S1 未建档提交快速建档 →S2 建档中
S2 建档中医生确认+有复诊 →S3 待复诊｜ 医生确认+无复诊 →S6-B 暂无安排
S3 待复诊到达当天 →S4 复诊当天｜ 过期未完成 →S5 逾期未复诊
S4 / S5扫码就诊 →正在就诊
正在就诊就诊结束+新复诊 →S3 待复诊｜ 标记无需复诊 →S6-A 无需复诊｜ 无新计划 →S6-B 暂无安排
S5 逾期医生确认改期 →S3 待复诊
S6-A / S6-B医生下发复诊计划 →S3 待复诊｜ S6-B 标记无需复诊 →S6-A 无需复诊
