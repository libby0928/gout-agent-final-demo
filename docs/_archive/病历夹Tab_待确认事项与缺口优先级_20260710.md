## 二、缺口优先级（供一期范围确认）

### P0（临床安全相关，建议一期就补字典 + 展示）
- 用药：停药原因 + 不良反应（关系用药安全）
- 用药：依从性细分 规律/偶漏/常漏（关系复发）
- 门诊：发作事件独立记录类型（关系"一年发几次"核心指标）

### P1（回看体验相关，建议一期展示、字典已有或易补）
- 门诊节点补：就诊医院/科室/医生 + 诊断结论
- 门诊节点补：医嘱 / 生活处方（字典需补独立字段或并入正文）
- 门诊 B 层待补：发作诱因、痛风石进展、手术/关节腔注射、在线问诊

### P2（长期价值，可后续迭代）
- 合并症时间轴联动
- 用药目标尿酸值属性（仅记属性不含趋势）

> P0 三项属字典级缺口，需先在字段字典 v1.1.1 回填「停药原因」「不良反应」「发作记录」等分类后才能对应扩展。P1 多为展示遗漏或 B 层字典补字段，可在不改动字典或小幅补字典的前提下直接落地。尿酸趋势 / 目标线 / 达标状态不纳入本双 Tab 字段梳理，留待检验趋势专题另议。
