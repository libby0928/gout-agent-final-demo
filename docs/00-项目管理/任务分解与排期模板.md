# 任务分解与排期模板（痛风智能体一期）

> 用法：复制本模板，填实际任务与状态。本文件是**唯一进度板**，不另建多套表。
> 状态枚举：未开始 / 进行中 / 待验收 / 已完成
> 卡片要素：编号 / 名称 / 域 / 端 / 验收标准 / 受约束规则 / 估点(人日) / 状态 / 本周优先级 / 依赖
> 排期节奏：确认锚点 + 双周滚动；单位"周"；关键路径=建档表单→医患关联→结构化档案展示。

## 双周滚动排期（示例锚点）

### 锚点前窗口（R-01/R-02 未确认）
- 做不依赖确认的项：5.1 结构化展示、4.1 OCR 归档、design-system 复用、字段字典补全。
- 禁做：1.1/1.2/1.4 表单定稿、8.2 路径依赖交互。

### 锚点后窗口（确认后）
- 解锁 1.x 建档表单、8.2 病史补充交互、3.2 录音回填判定。
- 关键路径串行推进：1.4 → 2.1 → 5.1 → 6.1 → 7.x。
