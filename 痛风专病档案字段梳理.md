# 痛风专病档案字段字典

## 1. 通用枚举

| 枚举名 | 值 |
|---|---|
| yes_no | yes、no |
| yes_no_unknown | yes、no、unknown |
| gender | male、female |
| source_type | patient_input、doctor_input、doctor_confirmed、system、ocr、audio_extract、his |
| confirm_status | draft、pending_confirm、confirmed、corrected、ignored |
| time_unit | day、month、year |
| attachment_type | joint_photo、tophus_photo、medication_photo、external_record、lab_report、other |

## 2. 通用字段

| field_key | 中文名 | 类型 | 必填 | 枚举/格式 | 规则 |
|---|---|---|---|---|---|
| patient_id | 患者 ID | string | 是 | - | 主键关联 |
| source_type | 来源 | enum | 是 | source_type | 字段级来源 |
| confirm_status | 确认状态 | enum | 是 | confirm_status | 字段级状态 |
| created_at | 创建时间 | datetime | 是 | ISO 8601 | - |
| updated_at | 更新时间 | datetime | 是 | ISO 8601 | - |
| confirmed_by | 确认医生 ID | string | 否 | - | confirm_status 为 confirmed/corrected 时可填 |
| confirmed_at | 确认时间 | datetime | 否 | ISO 8601 | confirm_status 为 confirmed/corrected 时可填 |

## 4. 痛风病史固定字段

| field_key | 中文名 | 类型 | 必填 | 枚举/格式 | 规则 |
|---|---|---|---|---|---|
| first_gout_pain_value | 首次痛风样疼痛时间数值 | integer | 否 | 正整数 | 与 first_gout_pain_unit 联合使用 |
| first_gout_pain_unit | 首次痛风样疼痛时间单位 | enum | 否 | day、month、year | 与 first_gout_pain_value 联合使用 |
| first_attack_site | 首次发作部位 | enum | 否 | ankle、foot_dorsum、first_mtp、knee、other | other 时填写 first_attack_site_other |
| first_attack_site_other | 首次发作其他部位 | string | 否 | - | first_attack_site=other 时填写 |
| has_tophus | 是否有痛风石 | enum | 否 | yes、no、unknown | - |
| comorbidity_options | 其他疾病/合并症 | array(enum) | 否 | diabetes、kidney_disease、stomach_disease、fatty_liver、hypertension、kidney_stone、hyperlipidemia、cardio_cerebrovascular_disease、none、other | none 与其他选项互斥；other 时填写 comorbidity_other_text |
| comorbidity_other_text | 其他疾病手写内容 | string | 否 | - | comorbidity_options 包含 other 时填写 |
| current_symptoms | 当前疼痛和红肿活动状态 | array(enum) | 否 | no_pain、mild_pain、obvious_pain、red_swelling_obvious、activity_limited | no_pain 与其他选项互斥 |
| urate_lowering_drug_status | 长期降尿酸用药情况 | enum | 否 | taking_over_6m、taking_under_6m、used_then_stopped、only_when_pain、never_used、unknown_if_urate_lowering | - |
| urate_lowering_drug_names | 降尿酸药物 | array(enum) | 否 | febuxostat、allopurinol、benzbromarone、other、unknown | urate_lowering_drug_status 为 taking_over_6m/taking_under_6m/used_then_stopped 时可填 |
| urate_lowering_drug_other | 其他降尿酸药 | string | 否 | - | urate_lowering_drug_names 包含 other 时填写 |
| urate_lowering_duration_months | 降尿酸药服用时长 | integer | 否 | 月 | 有服用记录时可填 |
| current_anti_inflammatory_meds | 当前消炎止痛用药 | array(enum) | 否 | glucocorticoid、nsaid、colchicine、unknown_name、none | none 与其他选项互斥 |
| attacks_last_year | 近 1 年发作次数 | enum | 否 | zero、one_to_two、three_to_five、more_than_five、unknown | - |
| gout_related_attachments | 痛风相关照片或资料 | array(file) | 否 | attachment_type | 可多文件 |

## 12. 互斥与条件规则

| rule_key | 规则 |
|---|---|
| comorbidity_none_mutex | comorbidity_options 包含 none 时，不允许同时包含 diabetes、kidney_disease、stomach_disease、fatty_liver、hypertension、kidney_stone、hyperlipidemia、cardio_cerebrovascular_disease、other |
| comorbidity_other_required | comorbidity_options 包含 other 时，comorbidity_other_text 必填 |
| current_symptoms_no_pain_mutex | current_symptoms 包含 no_pain 时，不允许同时包含 mild_pain、obvious_pain、red_swelling_obvious、activity_limited |
| anti_inflammatory_none_mutex | current_anti_inflammatory_meds 包含 none 时，不允许同时包含 glucocorticoid、nsaid、colchicine、unknown_name |
| first_attack_site_other_required | first_attack_site=other 时，first_attack_site_other 必填 |
| female_reproductive_condition | preparing_pregnancy、menopause、adverse_pregnancy_history、pregnancy_count、delivery_count 仅 gender=female 时有效 |
